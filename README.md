## Opera

Recomendo o Opera pela opção nativa de VPN.

1. Projeto
    - npm run build opera
2. Menu bar:
    - Extensions
    - Ative o modo Developer mode
    - Clique no botão Load unpacked
    - Escolha a pasta dist/opera
